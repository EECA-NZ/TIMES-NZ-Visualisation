EECA’s Energy End Use Database (EEUD) interactive visualisation tool.

Find out how energy is consumed across New Zealand with breakdowns of fuel type, sector, end use and technology. Information from EECA’s Energy End Use Database is presented here as an interactive visualisation tool. The tool allows users to view and explore how energy is used with interactive Sankey diagrams and bar charts.

In this page, hover over the Sankey diagram to see how different of fuel are utilised for different sectors and end uses.

In the Data Explorer page view and interact with the Sankey diagram, the time series stacked column chart, and the stacked column chart for deep exploration of the data. Filters on the left margin allow for selection of different parameters within the data.

Icon for selection Description
Sankey diagram – select sector and year filters and choose the data items you want to see from the left to the right of the Sankey (e.g., Fuel to End Use).
Times series stack column chart – select from the sector dropdown and choose the data item you want the columns to be stacked by (e.g., Fuel or Sector Group) for comparison across years.
Stacked column chart – select sector, year, the data item you want to stack the columns with (e.g., Fuel), and the axis grouping (e.g., End Use Group).

Note: The Data Explorer currently only provides stationary energy data. EECA is currently reviewing its EEUD transport data, therefore all transport data is excluded from the Data Explorer page.
